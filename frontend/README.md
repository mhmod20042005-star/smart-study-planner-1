# Smart Study Planner

A web-based system designed to help university students organize their academic workload and create an effective study schedule.

## Project Description

Smart Study Planner helps students manage courses, assignments, and exam deadlines by generating an organized weekly study plan.

## Main Features

- Add courses
- Add assignments and exam dates
- Generate a weekly study plan
- Track task completion
- View upcoming deadlines
